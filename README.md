# Spectrum IDR Backend & Frontend

A real-time audio spectrum visualizer with USD/IDR exchange rate style interface. Built purely with Web Audio API and HTML5 Canvas.

## Features
- **Real-time Audio Visualizer**: Murni menggunakan Web Audio API & HTML5 Canvas.
- **Multi-platform Streamer**: Proxy link audio YouTube & SoundCloud via backend `server.js` + `yt-dlp`.
- **Visualisation Options**: BASS, MID, FULL, WAVE, MAKS.
- **Invert Horizontal**: Opsi untuk membalik arah visualisasi (Bass Kanan).
- **Watermark Developer**: Link profil sosial media pengembang di dalam menu drawer.
